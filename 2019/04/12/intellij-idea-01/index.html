IDEA使用总结1—常用快捷键

发表于 2019-04-12 | 更新于 2019-04-14 | 分类于 Java工具 | 评论数： | 阅读次数：
本文字数： 1k | 阅读时长 ≈ 1 分钟

    以下是idea的常用快捷键总结，对于快捷键的使用应该是多上手，多用就记住了，不用刻意记住。

IntelliJ IDEA 常用快捷键(共33条）

1.搜索任何地方文件名
Shift 快速连续按两下

2.导入包，自动修正
Alt + Enter

3.新建或者生成代码
Alt + Insert

4.选中方法，查看参数信息
Ctrl + P

5.弹出当前打开的文件列表
Ctrl + E

6.根据0-9之间的数字，打开各个相应的工具窗口//
Ctrl + Num 0-9

7.保存文件
Ctrl + S

8.打开项目属性配置对话框
Ctrl + Shift + Alt + S

9.打开整个IDEA属性配置对话框
Ctrl + Shift + S

10.查找当前文件
Ctrl + F

11.快速定位到某一行某一列
Ctrl + G

12.替换
Ctrl + R

13.在路径里寻找
Ctrl + Shift + F

14.在路径里替换
Ctrl + Shift + R

15.环绕选中代码块，生成注释
Ctrl + Alt + T

16.注释代码或者取消注释
Ctrl + /或者ctrl + shift + /

17.格式化代码
Ctrl + Alt + L

18.优化导入，删除掉没用被使用的导入
Ctrl + Alt + O

19.缩进选中的行
Tab

20.取消缩进选中的行
Shift + Tab

21.将当前行剪切到剪贴板
Ctrl + X

22.将当前行复制到剪贴板
Ctrl + C

23.从粘贴板复制
Ctrl + V

24.重复输入当前行
Ctrl + D

25.删除当前行
Ctrl + Y

26.开始新的一行
Shift + Enter

27.选中的单词或者代码，切换大小写
Ctrl + Shift + U

28.返回或者前进最近编辑的地方
Ctrl + Alt + ←或者→
(按键冲突，我改为Shift + Q)

29.打开变量声明处
Ctrl + B或者 Ctrl + 鼠标左键

30.打开类型声明处
Ctrl + B

31.插入注释模板
Ctrl + J

32.代码提示
Ctrl + ,

33.补齐代码
Ctrl + Shift + Enter

常用的有fori/sout/psvm+Tab即可生成循环、System.out、main方法等boilerplate样板代码

不需要手动保存，idea会自动保存

在类中直接使用ALT+SHIFT+T就可以自动生成对应的Junit方法了
